Install
------------------------------------------------------------------------------

``sanhe_devops_tools`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install sanhe_devops_tools

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade sanhe_devops_tools
